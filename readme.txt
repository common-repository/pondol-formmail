== Description ==
This plugin is a form to email.
you can receive mail from web whe a user send a form.
Also this mail will be saved to database.
This include form generator which you create your own form.

== Installation ==
1. extract pondolplugin_formail.zip and Upload  to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to PondolePlugin Formmail and create new form mail
4. you will get a shortcode and post the shortcode where you want
